# 🧠 NeuroCast AI  
### 📄➡️🎙️ Turn PDFs into interactive learning experiences

---

## 🚀 Overview

**NeuroCast AI** is an AI-powered application that transforms PDFs into active learning sessions.

Instead of passively reading documents, users can:
- listen to them as podcasts
- understand them through summaries
- interact with a chatbot trained on the content

---

## 🎯 Problem

Students often struggle with:
- long and dense PDFs  
- low retention when reading  
- lack of interactive learning tools  

---

## 💡 Solution

NeuroCast AI turns static documents into:

- 🧠 **Summaries** for quick understanding  
- 🎙️ **Audio lessons** (podcast-style)  
- 💬 **Interactive chatbot** for questions  

---

## ⚙️ Features

- 📄 PDF text extraction  
- 🧠 AI-generated summaries  
- 🎙️ Text-to-speech podcast generation  
- 💬 Context-aware chatbot (based on uploaded content)  

---

## 🧩 Tech Stack

- **Python**
- **Gradio** (UI)
- **OpenAI API**
  - LLM (chat completions)
  - TTS (text-to-speech)
- **PyPDF**

---

## 🧠 How It Works

1. Upload a PDF  
2. Extract and process the text  
3. Generate a summary  
4. Convert the content into a podcast  
5. Chat with an AI tutor trained on the document
